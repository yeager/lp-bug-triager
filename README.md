# Launchpad Bug Triager

Launchpad bug triage tool with NLP-based classification.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install lp-bug-triager
```
